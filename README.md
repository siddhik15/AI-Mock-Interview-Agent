

# SmartHire – AI Mock Interview Agent

An intelligent mock interview application built using Python, Ollama, and Llama 3. The system simulates technical interviews, evaluates candidate responses using a Large Language Model (LLM), generates structured feedback, assigns scores, and produces detailed interview reports.

## Features

- AI-powered answer evaluation
- Technical interview simulation
- Automated scoring system
- Detailed feedback generation
- Interview history tracking
- Report generation
- Prompt Engineering based assessments
- Llama 3 integration via Ollama

## Tech Stack

- Python
- Ollama
- Llama 3
- REST APIs
- Prompt Engineering
- File Handling

## Use Cases

- Technical interview preparation
- Self-assessment and skill evaluation
- AI-powered candidate screening
- Educational and learning assistance
